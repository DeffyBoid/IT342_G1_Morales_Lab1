# Morales - User Registration and Authentication

## IT342 Lab 1 - Morales Group 1

## Project Description
This project implements user registration and authentication as part of the IT342 lab assignment.

## Contents
IT342_G4_Morales_Lab1
├─ /web
    - Frontend React application
├─ /backend
    - Spring Boot backend
├─ /mobile
    - (optional) mobile client
├─ /docs
    - Documentation and notes
├─ README.md
    - Project overview and instructions
└─ TASK_CHECKLIST.md
    - Project tasks and milestones

## Requirements
- Java 17+, Maven (for backend)
- Node.js 16+ and npm (for frontend)

## Installation (quick)
1. Backend: cd `backend` && ./mvnw package
2. Frontend: cd `web` && npm install

## Usage (quick)
1. Start backend: `cd backend && ./mvnw spring-boot:run`
2. Start frontend: `cd web && npm start`
